# PriTube Frontend

A modern music streaming and discovery platform built with React, Vite, Tailwind CSS, and Material UI.

## Features

* User Authentication
* Music Discovery
* Artist Search
* Album Browsing
* Playlist Management
* Music Recommendations
* Responsive Design
* Dark Theme UI
* Protected Routes
* User Dashboard

## Tech Stack

* React
* Vite
* React Router
* Material UI (MUI)
* Tailwind CSS
* Axios
* Context API

## Project Structure

```text
src/
├── api/
├── assets/
├── components/
├── context/
├── hooks/
├── layouts/
├── pages/
├── routes/
├── services/
├── utils/
└── App.jsx
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=
```

## Installation

```bash
git clone <repository-url>

cd frontend

npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## API Integration

All API requests are configured through:

```text
src/services/
```

Update the API base URL through the environment variable.
